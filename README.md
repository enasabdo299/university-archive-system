# 📚 University Archive System

نظام ويب متكامل لأرشفة وإدارة مشاريع التخرج والبحوث الجامعية، تم تطويره باستخدام **PHP** و **MySQL**.

---

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| PHP | لغة البرمجة الخلفية |
| MySQL | قاعدة البيانات |
| PDO | الاتصال الآمن بقاعدة البيانات |
| HTML5/CSS3 | واجهات المستخدم |
| JavaScript | التفاعلات والديناميكية |
| Chart.js | الرسوم البيانية والإحصائيات |
| Font Awesome 6 | الأيقونات |

---

## 👥 أدوار المستخدمين

| الدور | الصلاحيات |
|-------|-----------|
| 🎓 **طالب** | تسجيل حساب، رفع مشاريع، عرض الملف الشخصي |
| 📋 **موظف أرشيف** | مراجعة المشاريع (قبول/رفض)، تصنيف، تقارير |
| 👑 **مدير النظام** | إدارة جميع المستخدمين، صلاحية كاملة |
| 👁️ **زائر** | تصفح وبحث وتصفية المشاريع |

---

## ✅ الميزات المنفذة

- ✅ نظام مصادقة آمن (تسجيل دخول، تسجيل حساب، جلسات)
- ✅ نظام صلاحيات (RBAC) بـ 3 أدوار
- ✅ لوحات تحكم منفصلة لكل دور
- ✅ نظام رفع الملفات (PDF/DOCX) مع التحقق
- ✅ نظام بحث وتصفية متقدم
- ✅ تصميم متجاوب (Responsive)

## 🚀 كيفية تشغيل المشروع

1. نسخ مجلد المشروع إلى `htdocs` (عند استخدام XAMPP)
2. تشغيل Apache و MySQL
3. الذهاب إلى الرابط: `http://localhost/university-archive-system/setup_db.php`
4. تسجيل الدخول بالبيانات الافتراضية:
   - **Admin:** admin / 123456
   - **Archive:** archive / 123456
   - **Student:** student / 123456
